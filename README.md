# 🏦 Payoo Bank — DOM Project

A frontend banking web application built with HTML, CSS, and JavaScript, simulating core digital banking features through dynamic DOM manipulation.

**Live Demo:** [https://nihalxofficial.github.io/Payoo-Bank-Project/home.html](https://nihalxofficial.github.io/Payoo-Bank-Project/home.html)

---

## 📋 Features

- **Dashboard** — Displays the user's available balance at a glance
- **Add Money** — Deposit funds by selecting a bank (DBBL, IBBL, City Bank), entering an account number, amount, and 4-digit PIN
- **Cashout** — Withdraw money via an agent number with PIN verification
- **Transfer Money** — Send funds to another user account using account number and PIN
- **Get Bonus** — Redeem bonus coupons to add rewards to the account balance
- **Pay Bill** — Pay utility or bank bills by selecting a biller and entering payment details
- **Transactions** — View a log of all account activity

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Page structure and form elements |
| Tailwind CSS | Utility-first styling and layout |
| JavaScript (Vanilla) | DOM manipulation and interactivity |

> No JS frameworks — pure vanilla DOM-based interactivity with Tailwind for styling.

## 🚀 Getting Started

### Prerequisites
Just a modern web browser — no installation required.

### Run Locally

```bash
# Clone the repository
git clone https://github.com/nihalxofficial/Payoo-Bank-Project.git

# Navigate into the project directory
cd Payoo-Bank-Project

# Open the home page in your browser
open home.html
```

Or simply double-click `home.html` to open it in your default browser.
